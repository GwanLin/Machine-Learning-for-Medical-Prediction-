# Machine-Learning-for-Medical-Prediction-
Built a comparative prediction system using ML models on medical datasets (heart attack) 

•	Implemented multiple ML models including Logistic Regression, Random Forest, KNN, and Neural Networks, achieving high classification performance 
•	Conducted error analysis and model comparison, identifying trade-offs between interpretability, accuracy, and false negatives in medical predictions 
•	Performed statistical analysis on misclassified samples to improve model understanding and robustness
